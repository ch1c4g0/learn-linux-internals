
<p><h1>What is a directory?</h1></p>

<p>A special kind of file that contains links to other files. (Folder) is also a name for directory.</p>

<p><h1>What is a file?</h1></p>

<p>a self-contained piece of information available to the OS and any number of individiual programs.</p>

<p><h1>What is a filesystem?</h1></p>

<p>The way files are organized and managed, it can also refer to the type of format on a block device such as Ext4, XFS, Fat32, and NTFS.</p>

<p><h1>What is a Link?</h1></p>

<p>A link is a file that points to another file or directory(symbolic) or it can refer to a file that has more than one name (Hard-link)</p>

<p>What is a mount?</p>

<p>Refers to the way devices are made to appear in a unix/linux filesystem. Partitions, USBs, and CD/DVD are mounted to a directory somewhere in the filesystem.</p>
